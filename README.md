# garpun

## Config

```
node v12.13.1
npm v6.14.6
apt install vsftpd
apt install db_util
apt install sqlite3
npm install pm2 -g


```

## Instalation

```
1 npm update
2 Create .env file
  MEDIA_PATH= path where stored photos
  TRESH_PATH= path where stored file with wrong data type and file name
  API_SERVER=
  API_KEY=
  SQL_DB=
  PROVIDER=
```

##Start

```
npm start
or
npm run dev
```
